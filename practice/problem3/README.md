# Problem: Reverse Integer

<br>

Given a integer `n` reverse the integer.

<br>

**Example 1**

<pre>
<b>Input:</b> n = 123
<b>Output:</b> 321
</pre>

**Example 2**

<pre>
<b>Input:</b> n = -123
<b>Output:</b> -321
</pre>

**Example 3**

<pre>
<b>Input:</b> n = 120
<b>Output:</b> 21
</pre>

<br>

**Note: CANNOT CONVERT INT TO STRING!**
