# Remove The Numbers

<br>

Given a string `s`, remove all numbers in the string.

<br>

**Example 1:**

<pre>
<b>Input:</b> s = "abc123def"
<b>Output:</b> "abcdef"
</pre>

**Example 2:**

<pre>
<b>Input:</b> s = "123"
<b>Output:</b> ""
</pre>

**Example 3:**

<pre>
<b>Input:</b> s = "1abc2"
<b>Output:</b> "abc"
</pre>
